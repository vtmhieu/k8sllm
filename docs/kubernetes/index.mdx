# Kubernetes Core

Kubernetes là một distributed control system. Người dùng gửi desired state vào API server; controllers liên tục reconcile actual state về desired state; scheduler quyết định placement; kubelet thực thi trên node.

## Mental model cần giữ

- **API là contract**: mọi object, policy, và automation nên đi qua Kubernetes API hoặc một abstraction có lifecycle rõ.
- **Controllers là vòng lặp**: lỗi thường đến từ reconcile loop không idempotent, finalizer kẹt, hoặc dependency thiếu readiness.
- **Scheduler là capacity brain**: resource requests, taints, affinity, topology spread, và priority quyết định cluster có dùng tài nguyên đúng không.
- **Networking là policy surface**: Service discovery, ingress, egress, NetworkPolicy, DNS, và service mesh ảnh hưởng trực tiếp tới blast radius.
- **Storage là state contract**: StorageClass, PVC, backup, restore, và topology phải được review như data architecture.
